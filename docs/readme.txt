PARA COMPILAR:

gcc -Wall -o [nomeproexecutavel] [nomedasource].c

PARA EXECUTAR:

./[nomedoexecutavel] [param1] [param2] [param3] [param4]

param1 = Tipo de substituição (LRU, NRU, SEG);
param2 = Nome do arquivo com os endereços (pt.txt);
param3 = Tamanho da página. (em KB).
param4 = Tamanho total da memória.
